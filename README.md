# homebrew-codelima

Homebrew tap for [CodeLima](https://github.com/brianrackle/codelima).

## Install

```sh
brew tap brianrackle/codelima
brew install codelima
```

## Update

```sh
brew update
brew upgrade codelima
```

## Maintenance

`Formula/codelima.rb` is managed from the main `brianrackle/codelima` repository by the release workflow.
Tagged releases in the main repository publish the release assets and then update this tap automatically when the required GitHub Actions token is configured.
